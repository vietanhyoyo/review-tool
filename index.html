Coupang Review Scraper

Extension Chrome — không cần cài Python, không cần Terminal.

Bước 1 — Tải extension

⬇ Tải extension.zip

Phiên bản 1.0.0 · Chỉ yêu cầu quyền truy cập coupang.com

Bước 2 — Cài vào Chrome

  1. Giải nén file Giải nén extension.zip vào một thư mục bất kỳ.

  2. Mở trang extension Vào Chrome, gõ chrome://extensions vào thanh địa chỉ và nhấn Enter.

  3. Bật Developer mode Bật công tắc Developer mode ở góc trên bên phải.

  4. Load extension Bấm Load unpacked → chọn thư mục vừa giải nén.

Chỉ cần cài một lần. Extension sẽ xuất hiện trên thanh công cụ Chrome.

Bước 3 — Sử dụng

  1. Mở trang sản phẩm Coupang Đăng nhập Coupang và mở trang sản phẩm cần lấy đánh giá.

  2. Click icon extension Bấm vào icon Coupang Review Scraper trên thanh công cụ Chrome.

  3. Chọn số trang và bấm Lấy đánh giá File CSV sẽ tự động tải về — mở bằng Excel là xong.

Câu hỏi thường gặp

File tải về là CSV, mở trong Excel có hiển thị tiếng Hàn đúng không?
Có. File được lưu với UTF-8 BOM nên Excel trên Windows và Mac đều hiển thị đúng tiếng Hàn.
Extension có đọc dữ liệu cá nhân hay gửi dữ liệu ra ngoài không?
Không. Extension chỉ đọc nội dung trang Coupang đang mở và tạo file CSV ngay trên máy bạn. Không có server trung gian.
Tại sao phải đăng nhập Coupang?
Coupang dùng Akamai Bot Protection — chỉ trình duyệt thật đã đăng nhập mới vào được trang đánh giá.
Cài extension có an toàn không?
Extension chỉ yêu cầu quyền activeTab và scripting trên coupang.com. Bạn có thể xem toàn bộ code trong thư mục đã giải nén.
