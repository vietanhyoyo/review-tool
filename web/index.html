Chrome Session Bridge

Lấy review Coupang bằng session Chrome thật

Trang này không tự đọc cookie của Coupang. Thay vào đó, Python sẽ kết nối vào Chrome thật của bạn qua remote debugging và dùng chính session/cookie trong profile đó để scrape.

Hướng dẫn

  1. Bấm Mở Chrome — một cửa sổ Chrome riêng sẽ tự mở.
  2. Đăng nhập Coupang trong cửa sổ đó (chỉ cần làm 1 lần, lần sau tự nhớ).
  3. Quay lại đây, bấm Kiểm tra kết nối, rồi bắt đầu scrape.

Chrome thường của bạn vẫn mở bình thường — cửa sổ mới chạy song song.
